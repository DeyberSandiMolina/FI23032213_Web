## Respuestas a preguntas

- **Pregunta:** ¿Cuál carácter, además de la coma (,) y el Tab (\t), se podría usar para separar valores en un archivo de texto con el objetivo de ser interpretado como una tabla (matriz)? ¿Qué extensión le pondría y por qué?

  **Respuesta**
  - El punto y coma (;) es ampliamente usado como separador de valores, la extension es .csv, además de la coma (,) y el tabulador (\t), se puede usar el carácter pipe (|) para separar valores en un archivo de texto. La extensión recomendada sería .psv (Pipe Separated Values).
